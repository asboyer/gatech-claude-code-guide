Distributed Trace Correlator

2025

A backend service that ingests OpenTelemetry span streams over gRPC, builds causal DAGs of distributed traces in real time, and surfaces latency anomalies ranked by critical-path impact. Spans are partitioned into Kafka by trace ID to guarantee all segments land on the same consumer, then materialized into Redis with a TTL-bounded merge strategy before being flushed to Postgres for long-term querying.

Egocentric Action Anticipation

2025

A video understanding model that predicts the next action a person will take 1–5 seconds into the future from first-person wearable camera footage, trained on the EPIC-Kitchens benchmark. The architecture uses a dual-stream Vision Transformer — one branch encodes short-term optical flow deltas for motion context, the other encodes RGB frames for semantic context — with a cross-attention fusion head that learns which cues dominate at different prediction horizons.

  • Python
  • PyTorch
  • OpenCV
  • EPIC-Kitchens
  • Vision Transformer

Algorithm Complexity Visualizer

2025

A web tool that accepts Python algorithm submissions, parses them into an AST to instrument loop nesting depth and recursive call structure, then renders animated call graphs and live empirical time-complexity plots as the algorithm runs against input sizes from N=1 to N=10,000. Built for CS 3510 to give students a visual feedback loop between their implementation and its actual asymptotic behavior, with a sandboxed Docker execution environment for safe evaluation of untrusted code.

  • Flask
  • Matplotlib
  • NetworkX
